## 二、批次概述

### 2.1 目标

本批次实现 RAG 知识库系统的文档导入模块，包括：

1. **文档管理**：实现 Document 文档主表的 CRUD 操作
2. **版本管理**：实现 DocumentVersion 文档版本管理
3. **文件上传**：实现单文件和批量上传功能
4. **去重机制**：基于文件 Hash 实现自动去重
5. **导入任务**：实现 ImportTask 导入任务跟踪

### 2.2 范围

**包含：**

- Document 模型和 CRUD 接口
- DocumentVersion 模型和接口
- 文件上传接口（单文件/批量）
- 文件 Hash 计算和去重
- ImportTask 导入任务管理
- 文件类型自动识别

**不包含：**

- 文档解析（后续批次）
- 文件存储到 OSS（预留接口）

## 三、详细设计

### 3.1 数据库表设计

#### 3.1.1 documents 表（文档主表）

```sql
CREATE TABLE `documents` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '文档主键ID',
  `name` varchar(255) NOT NULL COMMENT '文档名称',
  `doc_type` varchar(50) NOT NULL COMMENT '文档类型：docx/doc/pdf/txt/md/html/xlsx/png/jpg/jpeg',
  `business_id` varchar(100) DEFAULT NULL COMMENT '业务归属ID',
  `business_name` varchar(100) DEFAULT NULL COMMENT '业务归属名称',
  `current_version_id` bigint DEFAULT NULL COMMENT '当前版本ID',
  `total_versions` int NOT NULL DEFAULT 1 COMMENT '版本总数',
  `status` tinyint NOT NULL DEFAULT 0 COMMENT '状态：0-待解析 1-解析中 2-已解析 3-解析失败 9-已删除',
  `total_pages` int DEFAULT 0 COMMENT '总页数',
  `total_chunks` int DEFAULT 0 COMMENT '总 Chunk 数',
  `creator_id` bigint DEFAULT NULL COMMENT '创建人ID',
  `creator_name` varchar(100) DEFAULT NULL COMMENT '创建人姓名',
  `created_at` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `updated_at` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  `is_deleted` tinyint NOT NULL DEFAULT 0 COMMENT '是否删除：0-否 1-是',
  PRIMARY KEY (`id`),
  KEY `idx_business_id` (`business_id`),
  KEY `idx_status` (`status`),
  KEY `idx_created_at` (`created_at`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='文档主表';
```

#### 3.1.2 document_versions 表（文档版本表）

```sql
CREATE TABLE `document_versions` (

  `status` tinyint NOT NULL DEFAULT 0 COMMENT '状态：0-待解析 1-解析中 2-已解析 3-解析失败',
  `parse_status` varchar(20) DEFAULT NULL COMMENT '解析状态：pending/processing/completed/failed',
  `parse_progress` int DEFAULT 0 COMMENT '解析进度（百分比）',

  `uploader_name` varchar(100) DEFAULT NULL COMMENT '上传人姓名',
  `uploaded_at` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '上传时间',
  `parsed_at` datetime DEFAULT NULL COMMENT '解析完成时间',
  `error_message` text COMMENT '错误信息',
  `created_at` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',

  KEY `idx_status` (`status`),

```
