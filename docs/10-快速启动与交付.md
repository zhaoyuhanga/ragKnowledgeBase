## 二、启动命令

### 2.1 方式一：Docker Compose启动（推荐）

#### Windows

```batch
# 1. 启动Docker服务
cd docker
.\start-docker.bat

# 2. 启动后端
cd ..
.\scripts\start-backend.bat
```

#### Linux/macOS

```bash
# 1. 启动Docker服务
cd docker
chmod +x start-docker.sh
./start-docker.sh

# 2. 启动后端
cd ..
chmod +x scripts/start-backend.sh
./scripts/start-backend.sh
```

### 2.2 方式二：本地环境启动

#### Windows

```batch
# 1. 初始化数据库
.\scripts\db-init.bat

# 2. 启动后端
cd backend
python -m uvicorn src.main:app --host 127.0.0.1 --port 8011 --reload
```

#### Linux/macOS

```bash
# 1. 初始化数据库
chmod +x scripts/db-init.sh
./scripts/db-init.sh

# 2. 启动后端
cd backend
python -m uvicorn src.main:app --host 127.0.0.1 --port 8011 --reload
```

### 2.3 方式三：Docker完整部署（生产环境）

```bash
cd docker
docker compose -f docker-compose.prod.yml up -d
```
