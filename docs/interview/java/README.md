2. 理解核心概念，掌握API使用

3. 分析常见问题及解决方案

1. Spring Boot高级 → Spring Cloud → Spring AI
2. 结合项目实践
3. 总结面试要点

## 面试重点

### 高频考点TOP15

1. **HashMap源码** - 底层结构、扩容机制、线程安全
2. **synchronized原理** - 锁升级、Monitor机制
3. **JVM内存模型** - 堆/栈/方法区、GC算法
4. **Spring IoC** - Bean生命周期、依赖注入
5. **MySQL索引** - B+树、聚簇索引、最左前缀
6. **Redis缓存问题** - 穿透、雪崩、击穿
7. **线程池原理** - 参数配置、拒绝策略
8. **Spring事务** - 传播行为、失效场景
9. **volatile/CAS** - 可见性、原子性
10. **设计模式** - 单例、工厂、代理
11. **Spring Boot自动配置** - @Conditional、条件注解
12. **Gateway网关** - 路由、过滤器、认证
13. **分布式事务** - Seata、AT/TCC模式
14. **Spring AI RAG** - 文档切分、向量检索
15. **链路追踪** - Sleuth、Zipkin

### 面试技巧

1. **原理讲解** - 不仅回答what，要解释why和how
2. **结合实践** - 用实际项目经验说明
3. **画图说明** - 数据结构、流程图能加分
4. **对比分析** - 不同方案优缺点要清晰

## 使用说明

### 题目格式

每道面试题包含：

- **题目类型**：基础概念/技术对比/技术原理/场景解决/架构设计

- **答案要点**：核心知识点梳理

1. 先通读一遍，建立知识体系
2. 重点题目重点记忆
3. 准备自己的项目经验佐证
