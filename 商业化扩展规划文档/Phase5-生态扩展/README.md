#### 2.2.2 SDK 设计

```python
# Python SDK 示例
from rag_qa import RAGQAClient

# 初始化客户端
client = RAGQAClient(
    api_key="your-api-key",
    base_url="https://api.rag-qa.com/v2"
)

# 知识库管理
kb = client.knowledge_bases.create(
    name="My Knowledge Base",
    description="Description"
)

# 文档上传
doc = client.documents.upload(
    kb_id=kb.id,
    file_path="/path/to/document.pdf"
)

# 问答
response = client.qa.ask(
    kb_id=kb.id,
    question="What is RAG?"
)

print(response.answer)
print(response.sources)
```

### 3.1 开放 API

# 认证
POST   /v2/auth/token                    # 获取访问令牌

GET    /v2/knowledge-bases              # 知识库列表
POST   /v2/knowledge-bases              # 创建知识库
GET    /v2/knowledge-bases/{id}       # 知识库详情
PUT    /v2/knowledge-bases/{id}       # 更新知识库
DELETE /v2/knowledge-bases/{id}       # 删除知识库

# 文档
POST   /v2/knowledge-bases/{id}/documents/upload  # 上传文档
GET    /v2/knowledge-bases/{id}/documents        # 文档列表
DELETE /v2/knowledge-bases/{id}/documents/{doc_id} # 删除文档

# 问答
POST   /v2/knowledge-bases/{id}/qa/ask  # 问答
