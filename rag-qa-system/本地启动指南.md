**版本：** V1.0
**日期：** 2026-05-13

## 1. 环境要求

### 1.1 硬件要求

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | 2 核 | 4 核及以上 |
| 内存 | 4 GB | 8 GB 及以上 |
| 磁盘 | 10 GB 可用空间 | 20 GB 可用空间 |
| GPU | 可选 | NVIDIA GPU (用于加速 Embedding) |

### 1.2 软件要求

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| Python | 3.12+ | 编程语言环境 |
| MySQL | 8.0+ | 元数据存储 |
| Redis | 6.0+ | 缓存服务 |

### 1.3 第三方服务

| 服务 | 必需 | 说明 |
|------|------|------|
| DeepSeek API | 是 | 需要从 [platform.deepseek.com](https://platform.deepseek.com/) 获取 API Key |
| sentence-transformers | 否 | 本地 Embedding 模型，无需 API |

## 2. 环境配置

### 2.1 克隆项目

```bash
# 克隆或下载项目到本地
cd d:/work/agent
```

### 2.2 创建虚拟环境（推荐）

```bash
# 进入项目目录
cd rag-qa-system

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

### 2.3 安装依赖

```bash
# 升级 pip
python -m pip install --upgrade pip

# 安装依赖
pip install -r requirements.txt
```

> **注意：** 首次安装时，`sentence-transformers` 需要下载模型（约 90MB），请确保网络连接正常。

### 2.4 安装 MySQL

#### Windows 安装 MySQL

1. 下载 MySQL Installer: https://dev.mysql.com/downloads/installer/
2. 选择 MySQL Server 8.0 安装
3. 配置 root 密码
4. 创建数据库：

```sql
-- 登录 MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE rag_qa CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### Docker 安装 MySQL

```bash
docker run -d \
  --name mysql-rag \
  -e MYSQL_ROOT_PASSWORD=root_password \
  -e MYSQL_DATABASE=rag_qa \
  -p 3306:3306 \
  -v mysql_data:/var/lib/mysql \
  mysql:8.0
```

### 2.5 安装 Redis

#### Windows 安装 Redis

1. 下载 Redis: https://github.com/microsoftarchive/redis/releases
2. 解压并运行 `redis-server.exe`

#### Docker 安装 Redis

```bash
docker run -d \
  --name redis-rag \
  -p 6379:6379 \
  redis:7-alpine
```

### 2.6 配置环境变量

```bash
# 复制环境变量模板
copy .env.example .env

# 编辑 .env 文件，填写以下配置：
```

编辑 `.env` 文件：

```env
# ==================== MySQL 配置 ====================
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_DATABASE=rag_qa
MYSQL_USER=root
MYSQL_PASSWORD=your_mysql_password

# ==================== Redis 配置 ====================
REDIS_HOST=localhost
REDIS_PORT=6379

# ==================== DeepSeek API 配置 ====================
DEEPSEEK_API_KEY=your_deepseek_api_key

# ==================== 其他配置 ====================
APP_ENV=development
DEBUG=true
LOG_LEVEL=INFO
```

## 3. 初始化项目

### 3.1 初始化数据库

```bash
# 方法一：使用 SQL 脚本
mysql -u root -p rag_qa < scripts/init_db.sql

# 方法二：使用 Python 脚本初始化
python scripts/init.py db init
```

### 3.2 下载 Embedding 模型

首次启动时，系统会自动下载 `sentence-transformers/all-MiniLM-L6-v2` 模型（约 90MB）。

如需手动下载：

```python
from sentence_transformers import SentenceTransformer
model = SentenceTransformer('sentence-transformers/all-MiniLM-L6-v2')
```

## 4. 启动项目

### 4.1 开发环境启动

```bash
# 确保 MySQL 和 Redis 已启动

# 启动应用
uvicorn app.main:app --reload --host 0.0.0.0 --port 8088
```

或使用 Python 直接运行：

```bash
python -m app.main
```

### 4.2 访问 API 文档

启动成功后，访问以下地址：

| 文档类型 | 地址 |
|----------|------|
| Swagger UI | http://localhost:8088/docs |
| ReDoc | http://localhost:8088/redoc |
| OpenAPI JSON | http://localhost:8088/openapi.json |

## 8. 常见问题

### 8.1 模型下载失败

**问题：** `sentence-transformers` 模型下载超时

**解决方案：**

```bash
# 设置镜像源
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple sentence-transformers
```

### 8.2 MySQL 连接失败

**问题：** `Access denied for user 'root'@'localhost'`

**解决方案：**

1. 检查 `.env` 中的 `MYSQL_PASSWORD` 是否正确
2. 确认 MySQL 服务已启动
3. 尝试使用 root 用户登录 MySQL 确认密码

### 8.3 ChromaDB 初始化失败

**问题：** `ChromaDB 初始化失败`

**解决方案：**

```bash
# 确保数据目录存在且有权限
mkdir -p data/chroma
chmod 777 data/chroma
```

### 8.4 DeepSeek API 调用失败

**问题：** `API Key 无效` 或 `请求超时`

**解决方案：**

1. 确认 API Key 正确且有效
2. 检查网络连接
3. 增加超时配置：`DEEPSEEK_TIMEOUT=120`

### 8.5 端口被占用

**问题：** `Port 8088 is already in use`

**解决方案：**

```bash
# 使用其他端口启动
uvicorn app.main:app --port 8001
```
